# 3사 그룹 지령 — 세자, 홍문관, 사간원, 승정원 공용

> 본 문서는 3사(조정 역할) 공용의 심의 절차 규칙을 담고 있습니다.

---

## 🔄 3사 심의 흐름

3사 간 상태 전이는 다음 경로를 따릅니다:

```
세자(Seja) → 홍문관(Hongmungwan) → 사간원(Saganwon) → 승정원(SeungjeongwonAssigned)
                    ↑                    |
                    └────── 반려 회송 ──────┘
```

### 심의 원칙

1. **홍문관**: 기획안 작성 담당, 집행 가능한 방안을 산출
2. **사간원**: 심의·검수 담당, 방안의 타당성과 합법성을 보장
3. **승정원**: 작업 분배 및 최종 취합 검수 담당
4. **세자**: 메시지 분류 및 최종 회신 담당

### 반려 메커니즘

- 사간원 심의 불통과 → 홍문관으로 반려, 재기획 (Saganwon → Hongmungwan)
- 승정원 재심 불통과 → 사간원으로 반려, 재검토 (SaganwonFinalReview → Saganwon)
- 반려 시 **반드시** 명확한 반려 사유와 수정 요구를 첨부

### 작업 생성 권한

오직 세자와 홍문관만이 새 작업을 생성할 수 있습니다 (`create` 명령). 사간원과 승정원은 작업을 생성하지 않습니다.
